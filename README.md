# 📝 To-Do List App

A simple command-line **To-Do List Application** built with Python.  
This app allows you to add, view, delete, and toggle the completion status of tasks, all stored in a JSON file for persistence.

---

## 🚀 Features
- ➕ Add new tasks
- 📋 View all tasks
- 🗑️ Delete tasks by number
- ✅ Toggle task completion
- 💾 Persistent storage using `tasks.json`

---

## 📂 Project Structure
├── tasks.json        # Stores your tasks (auto-created if missing)
└── main.py           # Main application script
